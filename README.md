Analysis of biomarker-driven treatment response in oncology patients using TCGA data (clinical + gene expression)

# Biomarker-Guided Treatment Response Analysis in Oncology Patients

## Objective
To identify biomarkers and patient characteristics that predict treatment outcomes in lung cancer patients.

## Data Source
- TCGA (via cBioPortal)
- Clinical data + mRNA expression data

## Tools Used
- Python (Pandas)
- SQL (planned)
- Tableau (planned)

## Key Questions
- Which biomarkers predict better survival?
- Which patient subgroups respond better to treatment?

## Status
Data cleaning and preprocessing in progress.
